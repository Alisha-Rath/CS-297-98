CS297 Proposal

AI-Powered Legal Decision Support System

Alisha Rath (alisha.rath@sjsu.edu)

Advisor: Dr. Chris Pollett

Description:

The aim of this project is to develop an AI-powered legal decision support system using machine learning and large language models (LLMs). The system will be designed to predict legal case outcomes and sentencing based on initial filings, court documents, and transcripts. The first phase will involve developing a basic classification model using publicly available legal data and enhancing this model by integrating LLMs to extract detailed insights from legal documents for more accurate predictions.
